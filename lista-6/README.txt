A implementação da sexta lista foi realizada utilizando a linguagem de programação Java. Os arquivos enviados correspondem a um projeto criado na ferramenta de desenvolvimento Eclipse.

Foram concluídas todas as questões. As classes referentes a implementação estão no pacote "br.unb.graph". A classe Graph representa o Grafo em si. Através dela é possível inserir nós e arestas, realizar buscas (Largura e Profundidade), obter uma lista de nós que representa a ordenção topológica, e por fim, obter uma lista de nós que simboliza o menor caminho entre dois nós (Algoritmo de Dijkstra).

No pacote "br.unb.test" são realizados alguns testes de impressão na classe "TestGraph". Os exemplos usados nos testes são baseados nas ilustrações apresentadas nos slides da disciplina.
